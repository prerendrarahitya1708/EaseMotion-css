# GSAP Theme Toggle Animation

A smooth, GSAP-powered theme toggle button with spinning and scaling micro-animations for switching between light and dark modes.

## Description

This component adds a high-quality micro-interaction to a standard theme toggle button using GSAP. When clicked, the button container spins 180 degrees while popping in and out, and the active sun/moon icon gracefully fades and scales up into view.

## Features
- **GSAP Powered:** Utilizes `gsap.fromTo` for fluid timeline control and easing.
- **Micro-interactions:** Rotation, scaling, and opacity changes all happen in tandem for a premium feel.
- **Easy to use:** Just drop in the HTML/CSS and ensure GSAP is loaded on the page.

## Usage

1. Include GSAP in your `<head>` or before the script:
   `<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/gsap.min.js"></script>`
2. Open `demo.html` in your browser.
3. Click the toggle button to see the animation in action.
